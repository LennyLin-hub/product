# 注塑件加工厂生产流程

本文档用于说明注塑生产的业务流程边界，便于与系统中的计划、派工、执行和完工状态对应。

> 简化范围：合并后处理、检验、入库流程，去除预处理。

## 接单与评审（订单输入）
- 接收客户订单：产品型号/图纸版本、数量、交期、包装要求、质量要求
- 内部评审：确认模具是否可用、材料是否满足、产能是否满足交期
- 输出：订单状态、产品工艺路线模板（用于后续计划与排程）

## 物料与工装准备（备料/备模）
- 物料准备：原料（塑料粒子）、色母/色粉、添加剂、包材
- 库存管理：库存查询、领料/扣减、物料批次记录（用于追溯）
- 模具准备：模具编号、状态（可用/保养/维修）、备件确认
- 输出：备料完成标记、模具可用标记、物料/模具批次信息

## 上机与换模（调机/首件确认）
- 换模：装模、接水路/电加热等辅助系统，安全确认
- 参数设定：温度、压力、保压、冷却时间、注射速度等
- 首件确认：首件试模与检验，确认OK后放量生产
- 输出：换模记录、首件确认记录、设备占用状态（进入生产）

## 注塑成型（核心生产）
- 循环生产：合模 → 注射 → 保压 → 冷却 → 开模 → 顶出取件
- 过程记录：产量、良品/不良品数量、停机原因/异常
- 输出：生产报工（按时间段/批次/任务）、质量与异常数据

## 后处理&检验&入库（合并工序）
- 后处理：修边/去披锋、简单二次加工（如需）
- 检验：抽检/终检，记录缺陷类型与处置（返工/报废/放行）
- 入库与包装：成品打包、贴标、入库，绑定批次信息（料批/机台/模具/时间段）
- 输出：完工反馈、入库记录、可追溯的成品批次档案

## 出货
- 出库发运：按订单生成出库/出货单
- 对外交付：物流信息、对账信息（如需）
- 输出：订单交付完成状态、出货记录（用于追溯与统计）
